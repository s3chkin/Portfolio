Freelancer
A web application for searching and offering services. The site allows posting and accepting ads, creating requests, and communication with the administrator. The seller can see the buyer's progress, but only if the buyer marks it in the ad itself.
Progress is indicated in percentages. The buyer has the right to select additional services from comparison tables, depending on the price. Revisions, delivery time, and other details depend on the price.

Full Travel
A web application designed to connect passengers seeking shared rides between cities with drivers offering available seats in their vehicles. Through FullTravel, users can easily find a suitable shared ride or post their own listing for a shared trip.
The web application provides a convenient and secure way to organize intercity transportation, reducing costs and unwanted emissions. With FullTravel, users can save time, money, and contribute to a cleaner environment by sharing rides with others.
